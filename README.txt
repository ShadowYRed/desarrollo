MANUAL DE USO

Requisitos de instalación para Python

*Se debe ejecutar el archivo de instalaciones.txt en el cual esta la dependencia necesaria
    la cual es Flask como unica libreria usada en el proceso
*cotizaciones.db es nuestro archiivo de base de datos de cual se consumira la información
*template/inex.html es el frontend del desarrollo
*app.py es nuestro archivo de backend donde gestionamos la conexion a la base de datos y 
    tambien la consulta y el update

Para la ejecucion del programa abrimos el archivo app.py y lo ejecutamos con f5, eligiendo Python
    como el motor con el que se ejecutara, en la consola nos mostrara la URL temporal con la que
    podemos abrir la pagina web en nuestro navegador (Esto solo se puede abrir en el equipo que este
    ejecutando el Python ya que es un servidor interno de pruebas)
